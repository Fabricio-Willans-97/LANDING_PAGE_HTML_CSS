O que você vai aprender voltado ao Front:

HTML - Abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. O HTML define o que vai em cima, o que é texto, o que é coluna, o que é imagem. Ele estrutura muito bem a página com início, meio e fim. Ou, se preferir, head, body e footer (cabeça, corpo e pé).

CSS - Cascading Style Sheets - é um mecanismo para adicionar estilo como cores, fontes, espaçamento, entre outras a um documento web. Em vez de colocar a formatação dentro do documento, o CSS cria um link para uma página que contém os estilos.

JavaScript - Uma linguagem de programação criada para ser parte dos navegadores web, para que scripts possam ser executados do lado do cliente e interajam com o usuário sem a necessidade deste script passar pelo servidor, controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido.

O que você vai aprender voltado ao Back:

TypeScript

TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem.

PHP

Uma linguagem de programação utilizada por programadores e desenvolvedores para construir sites dinâmicos, extensões de integração de aplicações e agilizar no desenvolvimento de um sistema.

MySQL

Um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. É atualmente um dos sistemas de gerenciamento de bancos de dados mais populares da Oracle Corporation, com mais de 10 milhões de instalações pelo mundo.
